# Jane App Branding Guide

Use these specs to update the Jane App booking pages to match the new website.

## Where to Update in Jane App

In your Jane App admin dashboard:
- **Settings → Online Booking → Appearance/Branding**
- Or **Settings → Customization**

---

## Brand Colors

| Element | Hex Code | Usage |
|---------|----------|-------|
| **Primary (Sage)** | `#7BA196` | Buttons, accents, links |
| **Primary Dark** | `#6a8f84` | Button hover states |
| **Navy (Text)** | `#2C3E50` | Headings, primary text |
| **Navy Light** | `#5a6c7d` | Secondary text, descriptions |
| **Background** | `#FDFBF9` | Page background |
| **Surface Alt** | `#f5f0eb` | Card backgrounds, sections |
| **White** | `#FFFFFF` | Cards, input fields |

---

## Fonts

| Type | Font Family | Weights |
|------|-------------|---------|
| **Headings** | Cormorant Garamond | 400, 500, 600, 700 |
| **Body** | Montserrat | 300, 400, 500, 600 |

**Google Fonts URL:**
```
https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@400;500;600;700&family=Montserrat:wght@300;400;500;600&display=swap
```

*Note: Jane App may not support custom fonts. If not, use their closest system font option.*
